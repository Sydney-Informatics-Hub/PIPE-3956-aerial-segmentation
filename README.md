# PIPE-3956-aerial-segmentation
Open source aerial segmentation models.


## Setup

### Local (or interactive VM)

```bash
conda create -n aerial-seg-dev python==3.9

conda activate aerial-seg-dev

pip install -r requirements.txt
```
